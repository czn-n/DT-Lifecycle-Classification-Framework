# Life Cycle-Oriented Digital Twin for Civil Infrastructure Assets: A natural language processing model-assisted review

This repository is associated with the manuscript:

"Life Cycle-Oriented Digital Twin for Civil Infrastructure Assets: A natural language processing model-assisted review"

The repository has been created to support the future public release of the resources developed in this study.

The following materials will be made publicly available upon acceptance/publication of the manuscript:

- SBERT-assisted lifecycle classification code
- Lifecycle keyword dictionaries
- Classification workflow documentation

The repository is currently under preparation while the manuscript is undergoing peer review.
